Autonomous Cars Website

Problems with Autonomous Cars Website

This is a project I made using HTML/CSS in October 2017 about the problems with Autonomous Cars. I made an intro page, a page discussing the problems the car faces on the road, a page discussing how Autonomous Cars can be hacked, and a sources page. On each page I implemented body text, images with captions and links to the image source, and links to the other pages. This was all done with HTML. CSS was used to align the text, align the images, choose the font and all the colors.
