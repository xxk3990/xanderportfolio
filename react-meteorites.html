React.js Meteorites Project

From mid 2022 to early 2023, a friend and I joined forces and improved our knowledge of React.js to create a project using it. We decided to create a web app that displays NASA's meteorite data (from a public external JSON file they have) to the user. The data was requested using fetch(). The app is written in React's FSC format and utilizes the UseState, UseMemo, and UseEffect hooks. After looking at some tutorials, we were able to implement pagination so the data is spread across multiple pages. Once pagination was working, we used Material UI's TextField element to create a search bar allowing the user to search for fallen meteorites. The user can also filter the meteorites by spans of 99 years starting with 800-899 (the first meteorite was in the year 861). The default (show all) is 700, as the filtering triggers once the slider is above 800. The pagination adjusts based on the filters and shows the amount of results for each filter. The mass value for the meteorites had all of them in grams, so we created a method to convert it to kilos and metric tons depending on the mass. The location value in the original JSON was coordinates only, so we decided to try and reverse-geocode it and were able to utilize the Geoapify reverse geocoding API to get the city and country based on the coordinates. The entire site is also mobile friendly!
