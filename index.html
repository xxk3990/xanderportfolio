About Me

My name is Xander Kaylan, and I've been writing code for several years now. Before RIT, I worked with JavaScript combined with HTML. I used this to make the Party List Using Google Firebase project. At RIT, I began doing JavaScript on its own, continued learning HTML, and started CSS. I used HTML/CSS to make many websites, including the Problems with Autonomous Cars project.
