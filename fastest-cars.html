Fastest Cars Project

In April 2021, I utilized HTML/CSS, JavaScript, and JSON to make a project detailing the fastest cars in the world as of April 2021. I began by doing some research on the fastest cars in the world, found a source that had a list (the source is cited in the sources page of the project), and put the information into a JSON file. Each entry in the JSON has a unique id (this allowed for only one details file – instead of one for each car, which I at first thought I'd have to do – that only displayed the information corresponding to the id), the make, model, top speed, description, and information for the source of the image. XMLHttpRequests are used to grab the data from the JSON and then display the properties on various pages. This project is also 100% responsive, however it achieves this in a way I had not used before: rather than resizing the original image in the CSS for each set of dimensions (what I was used to doing), for this project I was tasked with utilizing HTML's <picture> element, source media, and srcset attributes. What this entailed was duplicating each image a few times, resizing them for the different screen sizes, creating a separate JSON property for each image size, and using the <picture> element, source media, and srcset attributes to display a new image for each set of dimensions.

Note: The original article updated its list in April 2022 with some newer cars, though most of the ones from the previous list I used are still there. When I first made this project, all of the ones on this project's list were the ones in the article!
