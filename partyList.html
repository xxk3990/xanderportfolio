Party List W/ Google Firebase

Party List using Google Firebase

This is a website I made in November 2016 that allows the user to enter their information if they want to attend a party. Their data is then stored in a database, the one I used was Google Firebase. The user can add their info to the database and then display the list (their info along with the list of people already in the database). The user can also change their status using the checkbox. The default is "Invited", but if the user wants to attend they click the checkbox and it says "Change (Name + School) from invited to attending". I used JavaScript to save the info in the database, display it on the screen, and update the status. I used HTML to make the input boxes, the buttons, and the table containing the database list.
